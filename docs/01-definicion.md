# Nombre del proyecto: MyMusicRecord.
### Descripcion:
 Dashboard personal de Spotify que analiza tus hábitos musicales y los compara con las tendencias globales, para descubrir qué tan mainstream o alternativo es tu gusto.
### Dashboard
- Artistas mas escuchados.
- Canciones mas escuchadas.
- Los géneros musicales más escuchados.
- Comparacion con tendencias globales.
- Los horarios donde más uso Spotify.

## Stack tecnologico
- Frontend: React.js + Tailwind CSS
- API externa: Spotify Web API

## Por qué estas decisiones
Node.js permite levantar un backend rápidamente y que al compartir lenguaje con el frontend reduce la curva de aprendizaje
